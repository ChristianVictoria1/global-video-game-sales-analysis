# global-video-game-sales-analysis
Project Overview
- This analysis assesses global video game sales to identify top-performing genres, best-selling titles, and long-term sales trends within the gaming industry. The primary objective here is to uncover significant insights that could improve marketing and publishing decisions for game publishers and stakeholders.

Key Findings 
- The Action genre generates the highest global revenue among all video game genres.
- 2010 marked the peak year for global video game sales.
- North America contributes the largest share of global sales compared to other regions.
- Wii Sports is the best-selling video game globally, achieving the highest total sales.
